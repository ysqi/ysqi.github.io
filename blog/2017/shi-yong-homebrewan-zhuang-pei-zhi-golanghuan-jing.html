使用Homebrew安装配置golang环境

2017-02-17
编程语言与开发
约 202 字 预计阅读 1 分钟

文章目录

      • 安装Homebrew
      • 安装golnag

      • 安装完成

安装Homebrew

在Homebrew复制安装命令，在控制台运行完成安装

安装golnag

$ brew update && brew upgrade
$ brew install go

PATH配置

创建一个目录作为gopath,在目录创建三个目录bin、src、pkg

$ cd ~
$ vim .bash_profile

编辑.bash_profile文件并保存，文件内容如下

export GOROOT=/usr/local/opt/go/libexec
# GOPAT为上面创建的目录路径
export GOPATH=/Users/deweixu/coding/Go/go_path
export PATH=$PATH:$GOROOT/bin:$GOPATH/bin

运行source .bash_profile使配置的PATH生效。

安装完成

运行go env查看安装效果：

$ go env
